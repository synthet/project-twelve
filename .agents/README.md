# Antigravity / Gemini Setup

This directory (`.agents/`) serves as the Workspace Customizations Root for the Antigravity/Gemini agent environment.

## Architecture

- **Project Rules:** Antigravity reads workspace rules from `.agents/AGENTS.md`. This file is generated by `scripts/sync_assistant_trees.py` from the root `AGENTS.md`. Do not hand-edit it.
- **Repository Skills:** Antigravity automatically discovers skills inside `.agents/skills/`. These are generated from `.claude/skills/`.
- **MCP Configuration:** Antigravity loads Model Context Protocol servers from `.agents/mcp_config.json`.

## Setup

1. Copy the example configuration to your local config:
   ```powershell
   Copy-Item .agents\mcp_config.example.json .agents\mcp_config.json
   ```
2. Edit `.agents/mcp_config.json`:
   - Replace `RELAY_PATH` with the path to your Unity MCP relay (e.g., `%USERPROFILE%\.unity\relay\relay_win.exe`).
   - Replace `ABSOLUTE_PATH_TO_THIS_REPO` with the absolute path to this project checkout.

Keep secrets and machine-specific configurations out of version control. The `.agents/mcp_config.json` file is explicitly ignored in `.gitignore`.

## Ownership

- Author rules, commands, and skills under `.claude/`.
- Run `python scripts/sync_assistant_trees.py` to regenerate all mirrors, including `.cursor/`, `.agents/skills/`, and `.agents/AGENTS.md`.
