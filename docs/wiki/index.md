# Unity 2D Sandbox Knowledge Bundle

This directory is an Open Knowledge Format (OKF) bundle for the Unity 2D sandbox project. It follows the OKF convention of plain markdown files, YAML frontmatter on concept documents, progressive-disclosure `index.md` files, normal markdown cross-links, and an optional update log.

## Start Here

* [Project Brief](project-brief.md) - Vision, prototype scope, and non-goals.
* [Architecture Map](architecture-map.md) - Runtime layers, owners, and LLM editing guidance.
* [Roadmap and LLM Task Prompts](roadmap-and-llm-prompts.md) - Milestones and reusable prompt template.

## Systems

* [World and Chunk Data](world-and-chunk-data.md) - Chunk, tile, coordinate, and mutation conventions.
* [Rendering and Collision](rendering-and-collision.md) - Chunk-local rendering and collision direction.
* [Lighting, Liquids, and Simulation](simulation-systems.md) - Future tile lighting, liquids, and simulation budgeting.
* [Generation and Saving](generation-and-saving.md) - Procedural generation and save/load strategy.
* [Gameplay Systems](gameplay-systems.md) - Player, inventory, items, enemies, and pathfinding.
* [Multiplayer and Modding](multiplayer-and-modding.md) - Server-authoritative networking and data-driven content.
